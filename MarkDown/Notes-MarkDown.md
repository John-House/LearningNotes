# 链接
| 符号 | 含义 | 备注 |
| :- | :- | :- |
| \[文本](链接URL) | 内联式文本链接 | 英文标点符号 |
| \[文本][全局编号]<br><br>[全局编号]: 链接URL | 引用式文本链接 | 英文标点符号<br>全局编号是整个文档唯一的<br>文本和链接URL的声明之间至少空一行 |
| \!\[图片](链接URL) | 内联式图片链接 | 英文标点符号<br>以'!'开头 |
| \!\[图片][全局编号]<br><br>[全局编号]: 链接URL | 引用式图片链接 | 英文标点符号<br>以'!'开头<br>全局编号是整个文档唯一的<br>图片和链接URL的声明之间至少空一行 |
| 被脚注文本[\^编号]<br>[\^编号]: 脚注内容 | 脚注文本链接 | 编号并非整个文档唯一的<br>被脚注文本和脚注内容无需空一行 |

# 列表
| 符号 | 含义 | 备注 |
| :- | :- | :- |
| * 项目A<br>* 项目B<br>* 项目C | 无序列表 | *还可以用'+'或'-'代替<br>符号与项目内容之间必须至少有1个空格<br>使用缩进来表示列表的层级嵌套 |
| 1. 项目A<br>2. 项目B<br>3. 项目C | 有序列表 | 符号与项目内容之间必须至少有1个空格<br>使用缩进来表示列表的层级嵌套 |

- 项目A
- 项目B
- 项目C

# 文本样式
| 符号 | 含义 | 备注 |
| :- | :- | :- |
| \*文本\* | 斜体 | '*'与'文本'之间禁止存在空格 |
| \*\*文本\*\* | 加粗 | '**'与'文本'之间禁止存在空格 |
| \*\*\*文本\*\*\* | 斜体加粗 | '***'与'文本'之间禁止存在空格 |
| \~\~文本\~\~ | 删除线 | N/A |
| \<u>文本\</u> | 下划线 | N/A |
| \<kbd>文本\</kbd> | 键盘文本 | 例如<kbd>Ctrl+A</kbd> |
| \<sup>文本\</sup> | 上标文本 | N/A |
| \<sub>文本\</sub> | 下标文本 | N/A |
| | 换行 | 连续2个空格 |
<font color="red" size=5>示例：</font>
*斜体*
**加粗**
***斜体加粗***
~~删除~~
<u>下划</u>
<kbd>键盘</kbd>
